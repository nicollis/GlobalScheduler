GlobalScheduler
===============

The GlobalScheduler is a JavaFX application that allows users to schedule appointments with customers in different
countries and time zones. The application uses a MySQL database to store customer and appointment information.
The application also uses a login screen to authenticate users and track login attempts.

How to run the program
----------------------

1. Open the project in IntelliJ IDEA.
2. Press the green play button in the top right corner of the IDE.
Two configurations exist in the project to run the software
2a. Maven
2b. Application

Both run the application the same they just use two different methods from within the IDE.


Additional Report
-----------------

The additional report I chose to create was a report that shows all appointments for a selected customer.
